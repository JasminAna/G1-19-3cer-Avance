Formulario Home

El formulario Home es el formulario de entrada donde el usario puede visualilzar los una pagina de bienvenida, de igual forma contiene los diferentes enlaces que redirigen al usuario a los diferentes formularios, como: el formulario de estudiante, formulario de docente, formulario de asignatura y matricula donde cada formulario contiene diferentes funciones según los datos contenidos en cada formulario.

Formulario Estudiante

El formulario de estudiante contiene 5 funciones las cuales se desarrollaron en la base de datos PostgresSQL en la cual contienen diferente registros del sistema en su totalidad 10 registros de diferentes columnas como lo son: numero_alumno, nombre, apellido, fecha de nacimiento, dirección altura y carrera.

Dentro de las funciones que este formulario contiene estan:
GetAll la cual es un dato tipo get: Este nos llama la visualización de los datos ingresados en la base de datos.
GetOne la cual es un tipo de dato POST: Esta función nos permite visualizar los datos de un solo registro de la base de datos.
Insert la cual es un dato de tipo POST: Esta función nos permite ingresar nuevos datos a la base de datos de manera remota.
Update la cual es un dato de tipo Put: Esta función nos permite actualizar o modificar un registro previamente ingresado de maneraa remota.
Delete la cual es un dato de tipo DEL: Esta función nos permite elimar los registros no deseados de la base de datos de manera remota.
